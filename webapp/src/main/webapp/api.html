Accounts

Add Account

  • Description: Creates a new account.
  • Command: account/add
  • Parameters:
    • name: The name of the account
    • balance: The initial balance of the account
  • Response:
    {
   result: <0|1>,
   messages: [<message0>, <message1>, ...],
   content: {
      key: <accountKey>,
      name: <accountName>,
      balance: <accountBalance>
   }
}

Edit Account

  • Description: Updates an existing account
  • Command: account/edit
  • Parameters:
    • key: The key of the account
    • name: The new name of the account
    • balance; The new balance of the account
  • Response:
    {
   result: <0|1>,
   messages: [<message0>, <message1>, ...],
   content: {
      key: <accountKey>,
      name: <accountName>,
      balance: <accountBalance>
   }
}

Query Accounts

  • Description: Queries the server for a list of accounts.
  • Command: account/q
  • Parameters:
    • none
  • Response:
    {
   result: <0|1>,
   messages: [<message0>, <message1>, ...],
   content: [
      {
         key: <accountKey>,
         name: <accountName>,
         balance: <accountBalance>
      },
      ...
   ]
}

Cash Flow

Query Cash Flow

  • Description: Queries the server for information about the cash flow.
  • Command: cashflow/q
  • Parameters:
    • none
